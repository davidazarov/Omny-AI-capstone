# Agent Tools Documentation

This document outlines the deterministic Python functions exposed to the **Coach Agent**. These tools are used to perform mathematical calculations that Large Language Models (LLMs) often struggle to execute accurately.

By offloading these calculations to code, we ensure the fitness plans generated by Omny AI are biologically accurate and safe.

---

## 1. `calculate_bmr`

**Purpose:**
Calculates the user's **Basal Metabolic Rate (BMR)** using the **Mifflin-St Jeor Equation**.
This value represents the number of calories the user's body burns at complete rest to maintain basic life functions (breathing, circulation, cell production).

**Equation Logic:**
* *Male:* `(10 * weight) + (6.25 * height) - (5 * age) + 5`
* *Female:* `(10 * weight) + (6.25 * height) - (5 * age) - 161`

**Parameters:**

| Parameter   | Type    | Description                                      | Example |
| :---------- | :------ | :----------------------------------------------- | :------ |
| `weight_kg` | `float` | The user's body weight in Kilograms.             | `75.5`  |
| `height_cm` | `float` | The user's height in Centimeters.                | `180.0` |
| `age`       | `int`   | The user's biological age in years.              | `25`    |
| `gender`    | `str`   | Biological sex ("Male" or "Female") for scaling. | `"Male"`|

**Returns:**
* **Type:** `float` (or `int`)
* **Description:** The calculated BMR value in calories/day.
* **Example:** `1740.5`

## 2. `calculate_macros`

**Purpose:**
Determines the daily macronutrient targets (Protein, Fats, Carbohydrates) required to achieve the user's specific fitness goal.

**Logic:**
* **Protein:** Fixed at 2.0g per kg of bodyweight (High protein for muscle retention).
* **Fats:** Fixed at 0.8g per kg of bodyweight (Essential for hormonal health).
* **Carbs:** Variable multiplier based on `goal`:
    * *Lose Fat:* 2.0g per kg
    * *Maintain:* 3.0g per kg
    * *Build Muscle:* 4.0g per kg

**Parameters:**

| Parameter   | Type    | Description                                                                 | Example        |
| :---------- | :------ | :-------------------------------------------------------------------------- | :------------- |
| `weight_kg` | `float` | The user's body weight in Kilograms.                                        | `75.0`         |
| `goal`      | `str`   | The primary fitness objective. Must be: "Lose Fat", "Build Muscle", or "Maintain". | `"Lose Fat"` |

**Returns:**
* **Type:** `dict` (JSON Object)
* **Description:** A dictionary containing the specific gram targets for each macronutrient.
* **Example:**
    ```json
    {
      "protein": 150,
      "fats": 60,
      "carbs": 150
    }
    ```
